# Tecnologías

Estas son algunas de las tecnologías fundamentales utilizadas durante el aprendizaje y desarrollo de proyectos web.

---

## JavaScript (ES6+)

JavaScript es el lenguaje de programación que permite agregar lógica e interactividad a las páginas web.

Con JavaScript se pueden:

- Manipular elementos HTML.
- Responder a eventos del usuario.
- Consumir APIs.
- Gestionar datos.
- Crear aplicaciones web dinámicas.

Conceptos importantes:

- Variables
- Tipos de datos
- Condicionales
- Bucles
- Funciones
- Arrays
- Objetos
- DOM
- Asincronía
- ES6+

---

## HTML

HTML (HyperText Markup Language) es el lenguaje que define la estructura de una página web.

Ejemplo:

```html
<h1>Hola Mundo</h1>

<p>Este es un párrafo.</p>

<button>Click</button>
```

Con HTML se crean:

- Títulos
- Párrafos
- Formularios
- Botones
- Imágenes
- Enlaces
- Tablas
- Estructura general del sitio

### PDT

HTML define el contenido y la estructura.

---

## CSS

CSS (Cascading Style Sheets) permite dar estilos y diseño a una página web.

Ejemplo:

```css
h1 {
  color: purple;
  font-size: 32px;
}
```

Con CSS se pueden modificar:

- Colores
- Tipografías
- Tamaños
- Márgenes
- Posiciones
- Animaciones
- Diseño responsive

### PDT

HTML construye la estructura.

CSS se encarga de la apariencia visual.

---

## Git

Git es un sistema de control de versiones.

Permite:

- Guardar cambios en el proyecto.
- Volver a versiones anteriores.
- Trabajar en equipo.
- Llevar un historial de modificaciones.

Comandos básicos:

### Ver estado

```bash
git status
```

---

### Agregar cambios

```bash
git add .
```

---

### Crear un commit

```bash
git commit -m "mensaje descriptivo"
```

---

### Ver historial

```bash
git log
```

---

### Subir cambios

```bash
git push origin main
```

### PDT

Un commit representa una versión del proyecto en un momento determinado.

---

## GitHub

GitHub es una plataforma que permite almacenar repositorios Git en la nube.

Con GitHub se puede:

- Guardar proyectos online.
- Compartir código.
- Colaborar con otros desarrolladores.
- Mostrar proyectos en un portafolio.
- Gestionar versiones de una aplicación.

Flujo básico:

```text
Proyecto local
↓
git add .
↓
git commit
↓
git push
↓
GitHub
```
